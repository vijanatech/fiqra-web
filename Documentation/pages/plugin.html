Plugins

In this section, we are going to introduce Rocket LMS plugins that bring more power to your platform. You can purchase these plugins and add new features to your platform.
Payment gateway pack:
Rocket LMS includes 3 payment gateways by default (Paypal, Razorpay, PayU) but there are more payment gateways available that you can activate by purchasing the related plugin.
This plugin includes Stripe, Flutterwave, Paytm, Paystack, Zarinpal, Paysera, CashU, Yandex, Mercadopago, Bitpay, Midtrans, Iyzico, and Payfort integration that provides worldwide payments for your platform.
 Organizations:
This plugin enables organizational and multiple institutes education for your platform. You will be able to define different organizations with specific instructors and students.
Organizations will be able to create public or private classes. Private classes will be available just for organization students.
Offline Payment:
This option enables bank money transfer for your platform. The platform owner can define bank accounts from the admin panel and these accounts will be displayed for students during the payment process in the website frontend.
Users will be able to submit their payment information from the website frontend and the amount will be added to the user balance once the request is approved by the admin.
Subscription:
This plugin enables subscribe feature for your platform. You will be able to create different subscribe plans from the admin panel and users will be able to purchase subscribe packages and enroll on classes using subscribe plans.
Users will be able to view their subscription package statistics.
One of the most important points about this plugin is that lets you specify the classes that you want to sell them using this feature. In fact, you can enable subscriptions for specific classes.
Each subscription package could be limited by duration and subscribe count.
Quiz & certification:
This plugin enables quiz & certificates for your platform. Instructors can create different quizzes for their classes and each quiz could contain multiple-choice, descriptive, and image questions.
Each quiz could be time-limited or unlimited and you can also specify attempts for each quiz.
This plugin enables certificates with personalized templates and certificates will be generated automatically when a user reaches conditions.
Meeting Booking:
This professional plugin enables meeting booking feature. Instructors and organizations will be able to create meeting times and students will be able to reserve meeting sessions.
Instructors can define availability times for each day of the week and students will be able to book times using a calendar.
Instructors can specify their hourly charge and discount.
Text Courses:
This plugin allows instructors and organizations to create text-based courses.
Each text course includes sessions with text, image, or video content and the related files and students will be able to use this educational material and start learning.
SMS Authentication:
Rocket LMS uses email verification for the user sign-up process by default and this plugin enables SMS verification for the user sign-up process. They will receive the verification code by SMS.
Noticeboard:
This plugin enables a professional noticeboard for admin and organizations.
Admin will be able to post notices for all users or a specific group of them. Organizations also will be able to post notices for their students and instructors.
Bigbluebutton Integration:
Bigbluebutton is one of the best open-source software for conducting live classes. If you don’t want to use third-party live providers (Like Zoom), you can install BBB on a VPS and connect it to your website.
All of the live meetings will be created automatically on your BBB server.
